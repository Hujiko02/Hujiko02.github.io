# 隐私政策

Ohush 是由 胡济（以下简称"我们"）为您提供的白噪音应用。本隐私声明由我们为处理您的个人信息而制定。我们非常重视您的个人信息和隐私保护，将会按照法律要求和业界成熟的安全标准，为您的个人信息提供相应的安全保护措施。

## 一、我们收集的信息

Ohush 是一款纯离线应用，我们**不会**收集、存储或上传您的任何个人信息，包括但不限于姓名、手机号、位置信息、设备信息、使用记录等。所有音频播放均在本设备本地完成。

## 二、权限使用说明

- **后台运行权限**：用于在您退出应用或锁屏后继续播放白噪音，保障助眠体验。该权限仅在您主动启动播放时启用，停止播放后自动释放。

## 三、第三方服务

本应用**不接入**任何第三方 SDK、广告服务或数据分析工具。

## 四、信息存储与安全

您的设置偏好（如定时器时长）仅存储在设备本地，不会上传至任何服务器。我们不会也无法访问您的任何数据。
